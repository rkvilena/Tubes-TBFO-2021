Operan -> Operasi

Spesial -> '!' | '#' | '$' | '%' | '^' | '&' | '*' | '(' | ')' | '-' | '+' | '=' | '/' | '_' | '[' | ']' | '<'| '>' | '?' | ';' | '.' | ',' | '~' | '`' | '{' | '}' | '|'

Operator -> '-' | '+' | '*' | '/' | '/' '/' | '*' '*' | '%'
Angka -> Angka Digit | Digit

Spasi -> Spasi Spasi | '~'
SpasiAtauKosong -> Spasi | EEE

ListKarakterA -> Alfabet | '_'
ListKarakterB -> Alfabet | Spesial | Angka | '''
ListKarakterC -> Alfabet | Spesial | Angka | '"'

StringB -> ListKarakterB | StringB StringB
StringC -> ListKarakterC | StringC StringC

Desimal -> Angka '.' Angka

Konstanta -> Angka | Desimal | 'T' 'r' 'u' 'e' | 'F' 'a' 'l' 's' 'e' | '"' StringB '"' | ''' StringC ''' | Operasi
Subvar -> ListKarakterA Subvar | ListKarakterA | Angka Subvar | Angka
Var -> ListKarakterA Var | ListKarakterA Subvar | ListKarakterA

VarAtauKonstanta -> Var | Konstanta

Syarat1 -> SpasiAtauKosong Boolean SpasiAtauKosong

Syarat3 -> Syarat1 Spasi SubSyarat3

Syarat -> Syarat3 | Syarat4

Boolean2 -> VarAtauKonstanta
Boolean3 -> VarAtauKonstanta SpasiAtauKosong PembandingBoolean SpasiAtauKosong VarAtauKonstanta SpasiAtauKosong Operasi SpasiAtauKosong VarAtauKonstanta
Boolean4 -> VarAtauKonstanta SpasiAtauKosong Operasi SpasiAtauKosong VarAtauKonstanta SpasiAtauKosong PembandingBoolean SpasiAtauKosong VarAtauKonstanta

PembandingBoolean -> '<' | '>' | '<' '=' | '>' '=' | '=' '=' | '!' '=' | 'a' 'n' 'd' | 'o' 'r' | '|' | '&' | 'i' 's' | 'i' 'n'

ForA -> 'f' 'o' 'r' Spasi Var Spasi 'i' 'n'

While -> 'w' 'h' 'i' 'l' 'e' Spasi Syarat SpasiAtauKosong ':'

Break -> SpasiAtauKosong 'b' 'r' 'e' 'a' 'k' newLineAtauKosong BreakContinueEnding
Continue -> SpasiAtauKosong 'c' 'o' 'n' 't' 'i' 'n' 'u' 'e' newLineAtauKosong BreakContinueEnding 
BreakContinueEnding -> IsiLoop | SpasiAtauKosong

IsiLoop -> Break | Continue | S

newLine -> ';'
newLineAtauKosong -> newLine | EEE


HeaderIf -> If newLine IsiIf | If newLine IsiIf newLine HeaderElif | If newLine IsiIf newLine ELse newLine IsiIf 
HeaderElif -> Elif newLine IsiIf | Elif newLine IsiIf newLine HeaderElif | Elif newLine IsiIf newLine ELse newLine IsiIf

ELse -> 'e' 'l' 's' 'e' SpasiAtauKosong ':'

IsiIf -> S | HeaderIf | HeaderLoop


HeaderDef ->  Def newLine S | Def newLine Return | Def newLine S Return

Return -> SpasiAtauKosong 'r' 'e' 't' 'u' 'r' 'n' Spasi Return0
